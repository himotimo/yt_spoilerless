# yt_spoilerless

Watch YouTube videos without seeing the duration or progress bar — useful when you want to experience something without knowing how much is left.

Paste a YouTube URL or video ID, hit Load, and the video plays with all native controls hidden. A transparent overlay handles play/pause on click, and a fullscreen button appears on hover in the bottom-right corner.

## Running locally

```bash
npm install
npm start
```

Then open `http://localhost:3000`.
